## Overview

**Allen Choi**

*California Polytechnic State University San Luis Obispo*

This repository is where the work for my Master's thesis, cRab: A Novel Clustering Score using Clustering with Rivals and Buddies for Unsupervised Learning, is located.

### /thesis

This folder contains the files necessary to render the thesis paper itself, as well as the data needed to generate the visualizations and tables.

### /analyses

This folder contains the analyses generating the data used immediately in the thesis

### /drafts

This folder contains the miscellaneous analyses and drafts that were used in the development of the cRab score, as well as exploratory code used to find any insights related to its behavior.

### /data

This folder contains the miscellaneous data generated by the code in the /drafts folder.
